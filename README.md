# Secure Document Vault System

Secure Document Vault System is a cybersecurity-focused web application designed to protect sensitive documents through secure authentication, encrypted storage, and integrity verification.

The system allows users to register, log in, enable Multi-Factor Authentication using OTP, upload encrypted documents, verify file integrity using SHA-256 hashing, and manage files based on role-based access control.

## Key Features

- User registration and login
- Password hashing using bcrypt
- JWT-based authentication
- Multi-Factor Authentication with OTP
- Role-Based Access Control: Admin, Manager, User
- Secure document upload
- File encryption before storage
- SHA-256 hash integrity verification
- Digital signature verification
- Secure download and deletion
- HTTPS support using SSL certificates
- Admin panel for user role management
- Dark mode frontend using HTML, CSS, and JavaScript
